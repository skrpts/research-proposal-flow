## Purpose

A working checklist to ensure all ethical requirements are addressed before submitting an ethics application or beginning data collection.

## Ethics Application Checklist

### Study Design

- [ ] Research question clearly stated
- [ ] Study design appropriate for the question
- [ ] Sample size justified (power analysis for quantitative studies)
- [ ] Inclusion/exclusion criteria defined and justified

### Informed Consent

- [ ] Participant information sheet prepared
- [ ] Consent form prepared
- [ ] Information sheet written in accessible language (no jargon)
- [ ] Right to withdraw clearly stated
- [ ] Data usage explained clearly
- [ ] Contact details for questions provided
- [ ] Appropriate consent process for vulnerable participants (if applicable)

### Data Management

- [ ] Data management plan completed
- [ ] Lawful basis for data processing identified
- [ ] Data storage arrangements specified (encrypted, access-controlled)
- [ ] Anonymisation/pseudonymisation strategy defined
- [ ] Data retention period specified
- [ ] Data sharing arrangements documented (if applicable)
- [ ] Data Protection Impact Assessment completed (if required)

### Risk Assessment

- [ ] Potential risks to participants identified
- [ ] Mitigation measures for each risk documented
- [ ] Potential risks to researchers identified
- [ ] Distress protocol in place (if research may cause distress)
- [ ] Debriefing procedure documented (if deception is involved)
- [ ] Referral pathways identified (for participants needing support)

### Regulatory

- [ ] Ethics committee/IRB identified
- [ ] Application form completed
- [ ] All supporting documents attached
- [ ] Insurance/indemnity arrangements confirmed
- [ ] Supervisor approval obtained (if student research)
- [ ] Funder requirements checked

### Vulnerable Populations (if applicable)

- [ ] Enhanced DBS check obtained (if working with children or vulnerable adults)
- [ ] Gatekeeper consent arranged
- [ ] Age-appropriate materials prepared
- [ ] Additional safeguards documented
